前置工作：
1.安装前置库
pip install shapely
	# 如果你在虚拟环境中，请激活后再安装：
	source ~/your_ros_ws/devel/setup.bash
	pip install shapely
2.修改yaml文件，在工作空间内找到crazyswarmfly.yaml（大概是这个名字），里面只有一个无人机信息，需要改成两个：
crazyflies:
  - id: 1
    channel: 80
    initialPosition: [0.0, -0.5, 0.0]
    type: default

  - id: 2
    channel: 100
    initialPosition: [0.0, 0.5, 0.0]
    type: default

1.roscore
2.setup.bash
